Dynamic AI skill routing
via orbital mechanics.

Open source core

The MCP client is a 5 KB stdio shim under MIT. Same backend powers the browser miniapp and the npm package. One source of truth.

How it works

1

LLM authoring

Llama-3.3-70B (Workers AI) writes 5 candidate skills for your task — full markdown bodies, ~600 chars each, with named tools and decision rules.

2

Physics signature

For each skill we derive mass, scope, independence, cross_domain, fragmentation, drag, and dep_ratio — from content alone.

3

Class + score

Argmax over six per-class scores assigns the celestial class. Final ranking = lexical relevance × class boost × Lagrange versatility.
