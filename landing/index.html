How it works

1

Embedding pre-filter

An MiniLM encoder ranks all 73 skills by semantic similarity to your task and keeps the top 40.

2

Orbital scoring

Each candidate is scored against the task's profile using orbital-mechanics analogues — gravity wells, resonance, slingshot boosts.

3

Ranked output

You get back a ranked list with route_score, class, and a human-readable why string per skill.
