Reproduce

Every artifact in this dashboard is a single workflow_dispatch away.

  • HF Dataset · luuow/meridian-stellar-cache — raw + features + archetypes + harvest + specialists + evaluation + projection
  • GitHub · stellar/ — pull, features, archetypes, harvester, train_eval, project_jwst, plot_results
  • GitHub Actions — six workflows; each rerunnable on demand
  • Full write-up — methodology, honest gaps, what we didn't measure

What's NOT in this dashboard

  • No deployed harvester. No controller policy. The headline is forecast MAE, not energy delivered.
  • 3 stars is preliminary, not statistical. Cross-target claim needs ≥10 targets.
  • Cross-domain z-projection is a strong assumption — PSP and JWST aren't measuring the same physics.
  • At 12h–24h horizons, archetype-routing actively hurts. Method has a regime of validity, not universal lift.
