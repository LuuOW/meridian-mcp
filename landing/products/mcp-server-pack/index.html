    MCP Server Pack — 10 Production-Ready MCP Servers for Claude Code

    Ten single-purpose stdio MCP servers + a one-shot install script. Drop straight into ~/.claude.json. MIT-licensed.

    Most MCP server tutorials teach you how to write one tool. This pack gives you ten, already wired, already tested, already shipped. Calc, time, fs-search, HN search, arXiv, regex, JSON tools, markdown tools, Wikipedia, DNS — each as a single-purpose stdio server you can install in 30 seconds and call from Claude Code, Cursor, or any other MCP-aware client.

    calc

    dns-tools
